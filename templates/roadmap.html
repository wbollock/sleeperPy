SleeperPy

Planned Features

SleeperPy is free to use. This page tracks what is planned next and what is still exploratory.

Current Focus

  • Show league year badges to handle duplicate league names.
  • Tighten dashboard clarity for multi-league users.
  • Improve dynasty-specific insight accuracy.

Planned Next

  • Deeper waiver recommendations with lineup context.
  • Trade review history to show decision quality over time.
  • Cleaner weekly summaries for faster review on mobile.

Eventual Ideas

  • League trend tracking across multiple seasons.
  • Optional calendar reminders for key dynasty windows.
  • Expanded team need modeling for rookie drafts.

How priorities are set

Priorities are based on bug impact first, then features that help lineup and roster decisions the most each week.

Feedback is welcome via GitHub issues.
